Node（从百度开始）

网站 TJ

  • 官网
  • 中文官网
  • 社区
  • node 是从09年开始才有的
  • 深入浅出Node.js (朴灵 著)

Node 版本安装与下载

  • node以前就有两个版本 0.1 node.io(0.5)
  • node 4、6、8(偶数 稳定版本) 5、7、9（奇数 预览版本）
  • npm node package manager (node 包管理工具)
  • nvm node version manager (node 版本管理工具)
  • 推荐看英文版,
  • 最上面有个 DOWNLOADS,点击进入,Windows Installer (.msi)是直接进入安装,没有安装包,Windows Binary (.zip)时下载安装包,来安装
  • 如果要下载所有的版本,在All download options里查看,有历史所有的版本,下载有zip结尾的

nvm安装

  • 安装在c盘,任何英文软件不要有中文路劲,不要有空格,
  • 步骤 :
    1. 先配置nvm安装 路劲
      c:\node\nvm
    2. 再配置node安装路劲
      c:\node\nodejs
      再点下一步,

nvm使用,并切换安装nodejs版本

nvm 切换的版本都是node版本, cmd指令

  • nvm ls 查看有哪些node已经安装了
  • nvm install <版本号> 64/32 安装对应的node,如nvm install 6.9.5,64为系统可以省略写64,
  • nvm uninstall <版本号> 删除包
  • nvm use 版本号 切换包
  • node -v查看现在node的版本
  • 如果是通过下载安装包,要解压放到安装目录中,并改名为下载的版本号如v4.8.3
    安装目录如 ; C:\node\nvm.然后通过版本切换,换自己想要的版本

cmd指令 输入 node 编写代码

  • 如:var a=100 回车,
    var b=200 回车,
    a+b 回车,
    就可以输出结果,但不可以保存,
  • 但是可以在编辑器里建一个js文件写上jsd代码,然后打开这个jsm目录,shift鼠标右键打开cmd窗口,输入node 你的js文件名就可以输出结果,但node没有bom,dom ,window操作,注意写法
  • 浏览器页可以,不用输入node

REPL 可交互环境

    1. 通过cmd 命令可以直接使用node编写代码，但是这种直接编写不能很方便的保存
    1. 通过创建js文件，在js文件中编写js代码 然后通过 node app.js 运行你的代码

node工具 加载本地文件 ,服务器的方式加载 cmd指令(重点)

  • npm install -g http-server
  • 这时候会在你的版本文件夹那里多出4个文件,
    如v6.9.5下多出hs hs.cmd http-server http-server.cmd,还有node_modules文件夹下多出http-server文件夹,
    这是当前版本的如果要切换版本要重新再其他版本下载,
  • http-server 简写 hs
  • 这个工具用来将你当前的文件，以服务器的方式加载,
  • 你需要加载哪个文件夹就在哪个文件夹里shift加右键出cmd窗口,然后输入 http-server就会生成
  • 192.168.67.54:8081类的本地服务器,在浏览器输入192.168.67.54:8081,正个班都可以使用看都你的文件,
  • http-server -port 9000或者hs -port 9000 可以改端口别人就不可访问了,
  • http-server或者hs 重开端口
  • 如果你家的wife和你的手机在同一个局域网下,就可以访问
  • 如192.168.67.54:8081,的端口,这时候你的手机能访问,电脑也能访问,就相当与搭建一个小型服务器,语node环境运行的

node（服务器） 与浏览器不同

    1. 没有dom bom window
    1. this指向也不再指向window，空对象
    1. node可以直接读取本地文件
