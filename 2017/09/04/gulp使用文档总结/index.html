gulp使用文档总结

发表于 2017-09-04

gulp

1. gulp简介

  • gulp这个软件是用nodejs写的.
  • gulp是基于流的自动化构建工具
  • 网站开发完成之后,我们要做项目构建,完成之后才可以上线.
  • 项目构建:
    • 代码压缩.html css js
    • 代码混淆
    • 文件合并
    • 等其它自动化工作. sass转换为css.
  • 这就是gulp做的事情.

2. gulp安装

  • 首先需要全局安装gulp,如果之前已经全局安装过,可以省略这步.

    1. 还需要本地安装,在项目目录中.

3. gulp使用

    1. 在项目目录中新建1个gulpfile.js文件.

  • 在这个文件中写上构建代码.

  • 先引入本地的gulp模块

3.1 创建任务

  • gulp是以任务的形式来执行每一项构建化工作.

  • 调用gulp对象的task方法可以创建任务
    参数1: 任务名称
    参数2: 回调函数,执行该任务的时候,要做的事情.

  • 这个时候,我们就创建了1个任务testTask,该任务做的事情就是回调函数的事情.

3.2 执行任务

  • 如何执行这个gulp任务呢?

    1. 打开cmd工具,切换工作路径到项目目录, 是gulpfile.js`文件的目录

  • 在命令行窗口中使用gulp 任务名称就可以执行指定的任务.

  • 这个时候,名叫testTask的任务就会被执行.

4.压缩CSS gulp-cssmin

    1. 先下载 包 npm install gulp-cssmin –save

5.文件监视 watch

  • gulp对象提供了一个watch方法

  • 该方法的作用,监视指定文件的变化,

  • 一旦改动,就执行指定的任务

  1. gulp.task(“watchCss”,function () {
    //监视指定的css文件.可以使用通配符,一旦文件发送变化,就自动执行yscss任务
    gulp.watch(“./src/assets/css/reset.css”,[“yscss”]);

gulp.src(“./js/app.js”)
.pipe(uglify())
.pipe(gulp.dest(“./dist/js”));
